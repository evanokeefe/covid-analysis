Main Dataset:

The main dataset used for this project was the John's Hopkins COVID-19 tracker GitHub page using Confirmed Cases and Number of Deaths This dataset includes the total number of cases for each county within the United States (including territories which were exluded in this project). The dataset has a column for each date since January 22nd 2020 with the cummalitive sum number of cases.
For this project only the following columns were used:'Province_State', 'Admin2'(Name of the county), and 'date'.
